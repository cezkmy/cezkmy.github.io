LESSON 1

Introduction to Computing typically covers fundamental concepts in computer science and information technology. As a student, I learn about the basics of computer systems. This lesson also introduces key topics like computer hardware, software, networking, and the impact of computing on society.

LESSON 2

The professions and careers in computing offer diverse opportunities, including roles such as a software developer, network engineer, data scientist, cybersecurity analyst, systems analyst, database administrator, and IT project manager. It helps me determine which job best suits me in the future.

LESSON 3

The information processing cycle is a sequence of steps that computers follow to process data, including input, processing, output, and storage.

LESSON 4

A computer has four components: hardware, software, peopleware, and dataware. In this lesson, I learned a lot about what peopleware and dataware actually are. At first, I didn't know what peopleware and dataware were; I was only familiar with software and hardware. This lesson helped me learn more about computers.

LESSON 5

A number system represents numbers and is also known as the system of numeration. It defines a set of values to represent quantities. In the context of computers, there are four types of number systems: The decimal number system, Binary number system, Octal number system, and Decimal number system.

HTML AND CSS

HTML and CSS are scripting languages used to create web pages and web applications. HTML structures content, defining elements such as headings and paragraphs, while CSS handles presentation. I struggled a lot in this lesson because I don't have experience with HTML and CSS. Also, no one will teach me personally, so I rely on watching videos on YouTube.
